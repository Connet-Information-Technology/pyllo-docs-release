class pyllo.plugins.mainmenu.plugin.MainMenu(app: PylloApp)#

Bases: pyllo.core.runtime.spi.PylloPlugin

A Pyllo plugin to extend application with an aspect of features without a dockable widget(pane).

If you want to create a plugin that adds a new pane, please use PylloDockablePlugin.

New in version 1.0.0.
